カツ丼

材料（１人分）

  • 具材
    • とんかつ
    • 玉ねぎ：１／４
    • 麺つゆ
  • 卵：1個

作り方

  1. 具材を入れて火をかける
  2. 玉ねぎが透き通ってきたら、溶き卵を入れる
  3. 卵が半熟になったら火を止める
  4. ごはんの上にかけて完成

アレンジ

  • とんかつの代わりに揚物を入れてもOK

チャーシュー丼

材料（１人分）

  • チャーシュー：数枚（お好みで(
  • ソース
    • 刻み葱：適量
    • ニンニク（チューブ）：小さじ×２
    • ごま油：小さじ×１

作り方

  1. ソースを混ぜ合わせる
  2. ごはんの上にチャーシューを並べる
  3. ごはんの上にソースをかけて完成

アレンジ

  • チャーシューの代わりにハムでもOK

ドライカレー

材料（３人分）

  • 具材
    • ひき肉：２００ｇ（鳥肉が良い）
    • 玉ねぎ：１／２
    • 人参：１／３
    • ピーマン：２個

      • 人参、ピーマンはミックスベジタブルでもOK

  • 水：２００ＣＣ
  • カレールー：３片
  • ニンニク：チューブで少し

作り方

  1. 具材を炒める
  2. 火が通ったら水、カレールー、ニンニクを入れて煮込む
  3. 水分がなくなったら火を止める
  4. ごはんの上にかけて完成

アレンジ

  • ゆで卵があると良いです
  • キャベツの千切りの上にかけるのも良いです

ピラフ

材料（３人分）

  • 具材
    • 玉ねぎ：１個
    • ジャガイモ：１個
    • ミックスベジタブル：適量

  • 米：２合
  • 水：４５０ＣＣ（米３合だと５５０ＣＣ）
  • コンソメ：小さじ×３

作り方

  1. 具材を炒める
  2. 火が通ったら米を入れて軽く炒める
  3. 水とコンソメを入れる
  4. 強火で沸騰まで待つ
  5. 沸騰したら弱火で１５分
  6. 火を止めて１０分蒸らす

アレンジ

  • コンソメの代わりにカレールー×３片いれるとカレーピラフに
  • 具材にシーフードミックスを入れると、シーフードピラフに

土鍋で炊くごはん

材料（３人分）

  • 具材
  • 米：３合
  • 水：５４０ＣＣ（水の１．２倍）

    • 作り方

    1. 米をといで、水をいれて３０分まつ。（水は冷たいほうが良い）
    2. 土鍋に火をかけ、沸騰までは中火。（３号なら１０分程度）
    3. 弱火で１０分
    4. 最後に強火で１分（これで焦げができる）
    5. 火を止めて１０分蒸らし、米を切ってからさらに１０分蒸らす

    アレンジ

    • このレシピをベースに「混ぜご飯の素」を入れると、いろんな混ぜご飯が作れる
